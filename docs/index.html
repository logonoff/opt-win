Install

Install via Homebrew:

brew tap logonoff/superopt https://github.com/logonoff/superopt
brew install --cask superopt

Or download the latest release directly from GitHub Releases. The app is ad-hoc signed, so you may need to right-click and choose Open on first launch to bypass Gatekeeper.

Permissions

SuperOpt requires Accessibility and Input Monitoring permissions in System Settings > Privacy & Security. The app will prompt you on first launch.

Option Key Shortcuts

Single press ⌥

Open Mission Control instantly with a single press of the Option key.

Double press ⌥

Open Spotlight app search with a quick double press.

⌥+1-9

Launch the Nth app in your Dock. The Finder position is configurable.

GNOME-Inspired Features

Hot Corner

Move your mouse to the top-left corner of any screen to trigger Mission Control, complete with a GNOME-style ripple animation.

Home/End Keys

Home and End keys move the cursor to the start and end of the line, just like on Linux.

Middle-click Paste

Middle-click on a text field pastes at the click position (X11-style). Middle-click on a Dock icon opens a new window. Otherwise passes through for native behavior like open-in-new-tab in browsers.

More Features

Shortcut Remapping

Maps Linux keyboard shortcuts to macOS equivalents. Control+C/V/X/Z, browser shortcuts, terminal shortcuts, and more. Each shortcut can be individually turned on or off.

Caps Lock Indicator

Onscreen notification when Caps Lock is toggled, so you always know its state.

Scroll Zoom

Control+scroll zooms in and out in browsers, with natural or traditional scroll direction support.

Cut & Paste Files

Control+X then Control+V in the Finder moves files instead of duplicating them.

Window Tiling

⌥+Arrow keys to tile windows to the left or right half, maximize, or restore to their original size.

Green Button Fills

The green button fills the window instead of entering full screen. Click again to restore.

Dark Menu Bar

Shows a black bar behind the transparent menu bar when a window fills the screen.

Fine-Grained Control

Every feature can be individually toggled on or off in the settings window, down to individual keyboard shortcuts.

Fully Offline

No telemetry, no network requests, no data collection. Everything runs locally on your Mac.
