Install

Install via Homebrew:

brew tap logonoff/opt-win https://github.com/logonoff/opt-win
brew install --cask optwin

Option Key Shortcuts

Single press ⌥

Open Mission Control instantly with a single tap of the Option key.

Double press ⌥

Launch Spotlight Apps with a quick double tap.

⌥+1-9

Launch the Nth app in your Dock. Finder position is configurable.

GNOME-Inspired Features

Hot Corner

Move your mouse to the top-left corner of any screen to trigger Mission Control, complete with a GNOME-style ripple animation.

Home/End Keys

Home and End keys move the cursor to the start and end of the line, just like on Linux.

Middle-click Paste

Middle mouse button pastes from the clipboard, X11-style. Off by default.

More Features

Shortcut Remapping

Maps Linux keyboard shortcuts to Mac equivalents. Ctrl+C/V/X, Alt+F4, browser shortcuts, terminal shortcuts, and more. Each shortcut is individually toggleable.

Caps Lock OSD

On-screen notification when Caps Lock is toggled, so you always know its state.

Scroll Zoom

Ctrl+scroll zooms in and out in browsers, with natural or traditional scroll direction support.

Cut & Paste Files

Ctrl+X then Ctrl+V in Finder moves files instead of duplicating them.

Green Button Fills

The green window button fills the window instead of entering full screen. Click again to restore.

Dark Menu Bar

Shows a black bar behind the transparent menu bar when a window fills the screen.
